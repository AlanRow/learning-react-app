# Баг при перемещении элементов

В данной задаче намерено создан баг, который возникает если один из элементов списка пометить выполненным, а затемпопробовать поменять их местами

Воспроизведите баг и попытайтесь исправить, сделав копию данного репозитория

В решениии приложите коммит и объяснение решения
